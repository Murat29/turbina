# Turbina

### Описание
Cайт для музыкального проекта детского книжного магазиа Маршак. Родители присылают стихи детей, Маршак их отбирает и записывает музыкальные треки совместно с известными группами.

### Технологии
* HTML
* CSS
* JS
* React
